My Projects

Supplies Management System

A Supplies Management System is a digital tool designed to efficiently track, manage, and control inventory of supplies within an organization. It streamlines the procurement, usage, and replenishment processes, helping reduce waste, avoid stockouts, and maintain accurate records. This system improves overall operational efficiency by providing real-time visibility into supply levels and generating alerts for low stock or expiration dates.

Donation Management System

A Donation Management System is a software solution designed to streamline the process of collecting, tracking, and managing donations for non-profit organizations. It helps organize donor information, monitor donation campaigns, generate receipts, and provide detailed reports to ensure transparency and efficient fund allocation. This system enhances donor engagement and simplifies administrative tasks related to fundraising.

Ticketing System

A Ticketing System for technical assistance is a software solution designed to manage, track, and resolve IT support requests efficiently. It allows users to submit technical issues or service requests, automatically assigns tickets to the appropriate support personnel, and monitors progress until resolution. This system streamlines communication between users and support teams, prioritizes tasks, and ensures timely, organized handling of technical problems to improve overall service quality.

Ordering System

An Online Ordering System is a digital platform that allows customers to browse products, place orders, and make payments over the internet. It streamlines the purchasing process for businesses and enhances customer convenience by offering a fast, accessible, and user-friendly way to order anytime, anywhere.

Library System

The Library Management System is a desktop application developed using Java and Java Swing for managing book inventory, borrowing, and returns. It features a user-friendly graphical interface that allows librarians to add, update, and track books, manage member records, and monitor borrowing history efficiently. The system ensures organized library operations and provides a smooth experience for both administrators and users.
